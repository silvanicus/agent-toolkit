# English Editor

Reviews and improves English text for grammar, clarity, and tone while preserving the original intent. Tailored for non-native English speakers, especially Spanish speakers.

## Usage

```
/english-editor "Hi team, I wanted to share with you that the deploy of last friday was succesful."
```
