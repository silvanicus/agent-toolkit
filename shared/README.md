# Shared

This folder contains sample reference files that skills and agents read at runtime to personalize their output.

Think of it as a lightweight knowledge base about your organization. The more accurate these files are, the more relevant and specific the output from each skill will be.

---

## How to use these files

These are **templates with placeholder values**. To get the most out of the skills:

1. **Copy the file** or edit it directly. You can also add it as a PDF inside of a project. 
2. **Replace the placeholder content**: this are just templates and placeholders. 
3. **Keep it concise** — agents read these files at invocation time. Dense, specific information works better than long prose.

You don't need to fill everything in. Partial context is still useful.
